# **Python – Code Exercises and Utility Programs**

This folder contains a collection of Python scripts developed for learning, practicing, and demonstrating programming fundamentals. The files include small utility programs, assignments, and practice exercises focused on calculations, user input, conditional logic, and basic algorithmic thinking.

Below is an overview of the main scripts included in this folder:

### **BMI Calculator**

A program that calculates a user’s Body Mass Index based on height and weight. It evaluates whether the BMI falls into categories such as underweight, normal weight, overweight, or obese. This script demonstrates numerical operations and conditional statements.

### **Rectangle Area Calculator**

A simple utility that computes the area of a rectangle based on user-provided length and width. It illustrates variable handling, mathematical formulas, and working with user input.

### **Savings Calculator**

A practical program that asks the user to enter their monthly income and expenses. Based on the remaining amount, it determines whether the user is saving enough. This script showcases conditional logic, variable comparison, and basic financial evaluation.

### **Additional Python Scripts**

The folder also contains various other exercises and homework assignments. These include practice with loops, functions, input validation, and general programming tasks. Each script focuses on building core Python skills and improving problem-solving abilities.
