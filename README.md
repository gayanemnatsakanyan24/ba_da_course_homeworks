# **Data & Analytics Portfolio**

Welcome to my Data & Analytics portfolio.

This repository contains projects, exercises, dashboards, and learning materials across multiple analytics tools and programming environments.  
 Each folder has its own dedicated README describing its contents in detail.

---

## **Repository Structure**

### **1\. Excel**

This folder contains an Excel-based schema/model for a warehouse database.This spreadsheet defines the schema of a warehouse database. It contains the core tables needed to model inventory, products, categories, suppliers, and transactions. 

---

### **2\. Looker Studio**

This folder contains a dashboard built on the Classic Models database, visualizing business performance metrics using Google Looker Studio.

---

### **3\. Python**

This folder includes Python homework files, code exercises, and small utility programs such as:

* A BMI calculator

* A rectangle area calculator

* A savings calculator that evaluates whether your monthly savings are sufficient based on income and expenses

---

### **4\. SQL**

This folder contains SQL queries created for practice and data exploration.  
 The queries include SELECT statements, joins, aggregations, and other SQL fundamentals.

---

### **5\. Tableau**

This folder includes a Tableau dashboard based on the Microsoft Northwind database.  
 The dashboard focuses on sales insights, product performance, and regional analysis.

---

## **Purpose of This Repository**

This repository demonstrates skills in:

* Data modeling

* Dashboard design and analytics

* SQL querying

* Python scripting

* Data visualization (Looker Studio and Tableau)

* Working with real-world datasets and classic databases such as Northwind and Classic Models

It also serves as a clean portfolio for showcasing work to employers and collaborators.
